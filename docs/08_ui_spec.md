# 08 — UI 交互规格

> 对应原型图目录：`../prototype/screens/`
> 原型查看器：`../prototype/index.html`

---

## 全局布局

- 左侧导航栏（260px 宽）：消息、联系人、传输中心、共享文件、设置
- 右侧内容区：根据导航切换显示对应页面
- 顶栏：应用名 + 当前在线状态指示
- 最小窗口尺寸：900×600

---

## S01 — 首次启动向导

**页面用途：** 用户首次运行时引导设置。

**组件：** 昵称输入框、网络发现开关（toggle）、下载目录选择（文件夹浏览器）、"上一步"/"下一步"/"完成"按钮。

**行为：**
- 第 1 步：昵称输入。默认填充系统主机名。输入为空时"下一步"禁用。
- 第 2 步：网络发现开关。默认开启。点击 toggle 切换。
- 第 3 步：下载目录。显示当前路径。点击"浏览"打开系统文件夹选择对话框。
- 点击"完成"：值写入 SQLite，`onboarding_completed = 1`，关闭向导，打开主窗口。
- 中途关闭窗口：设置不保存，下次启动重新进入向导。

---

## S02 — 主界面（消息与会话）

**页面用途：** 默认打开的消息列表页。

**组件：** 会话列表（左侧）、聊天区域（右侧，空态时显示"选择一个会话开始聊天"）。

**状态规则：**
- 空态：显示"暂无会话，点击左侧联系人在线设备开始聊天"
- 有会话：按最后消息时间降序排列，未读会话加粗 + 未读计数红点
- 选中会话：右侧显示聊天窗口

---

## S03 — 单聊窗口

**页面用途：** 与一个设备的一对一聊天。

**组件：** 对方姓名+在线状态（顶部）、消息列表（中部，滚动）、输入框+发送按钮（底部）、文件拖拽区域。

**行为：**
- 回车或点击发送 → 文本消息发出，输入框清空
- 输入框为空时发送按钮禁用
- 拖拽文件到聊天区域 → 自动发起传输请求
- 消息列表：自己的消息靠右（蓝色气泡），对方消息靠左（灰色气泡）
- 系统消息居中灰色小字

**状态展示：**
- 发送中：消息气泡右侧显示旋转加载图标
- 发送成功：显示对勾
- 发送失败：显示红色感叹号，点击可重发
- 对方离线：输入框上方提示"对方当前离线，消息可能无法送达"

---

## S04 — 广播通知

**页面用途：** 发送广播消息给全网段。

**组件：** 消息输入框、"需要确认收到"复选框、发送按钮、已读/未读列表。

**行为：**
- 输入消息 → 点击发送 → 所有在线设备弹出通知弹窗
- 勾选"需要确认收到" → 接收方弹窗包含"已收到"按钮
- 发送方可查看确认统计（5/10 已确认）

---

## S05 — 传输中心

**页面用途：** 所有传输任务的统一视图。

**组件：** 标签切换（全部/进行中/已完成/失败）、任务列表、搜索框。

**行为：**
- 每个任务行显示：文件名、大小、对方昵称、方向箭头（↑↓）、状态、进度条（进行中）
- 点击已完成任务 → 打开文件所在目录（QDesktopServices::openUrl）
- 点击失败任务 → 显示失败原因 tooltip

**状态规则：**
- 空态："暂无传输任务"
- 进行中过滤为空："没有正在进行的传输"

---

## S06 — 在线共享（浏览）

**页面用途：** 浏览其他在线设备共享的文件。

**组件：** 共享源列表（左侧）、文件浏览器（右侧，含面包屑导航）。

**行为：**
- 共享源列表显示所有 `share_enabled = true` 的在线设备
- 点击设备 → 请求共享目录列表
- 首次访问某设备共享 → 对方弹出授权弹窗（S08）
- 文件浏览器支持点击目录进入子目录、点击文件触发下载

---

## S07 — 我的共享

**页面用途：** 管理自己共享的目录。

**组件：** 共享目录列表、"添加共享"按钮、每个共享项有启用/禁用 toggle 和删除按钮。

**行为：**
- 点击"添加共享"→ 系统文件夹选择对话框 → 选择后立即生效
- 关闭 toggle → 该共享在网络上不可见但不删除本地配置
- 删除 → 确认弹窗 → 从列表中移除

---

## S08 — 授权弹窗

**触发条件：** 其他设备首次请求访问我的共享文件。

**组件：** 请求者昵称+设备名、"记住我的选择"复选框、"允许"/"拒绝"按钮。

**行为：**
- 弹窗模态，阻塞共享访问直到用户选择
- 点击"允许"→ 对方可浏览，记录写入 access_grants
- 勾选"记住"→ 同设备后续访问不再弹窗
- 点击"拒绝"→ 对方收到 ACCESS_DENIED 错误

---

## S09 — 文件投递箱列表

**页面用途：** 查看和管理的投递任务。

**组件：** 投递任务列表（任务名、截止时间、提交数）、"新建投递"按钮。

**行为：**
- 点击投递任务 → 查看提交详情（提交者、文件名、时间、勾选状态）
- 点击"新建投递"→ 跳转到 S10

---

## S10 — 新建投递任务

**页面用途：** 创建新的文件收集任务。

**组件：** 任务名输入框、描述输入框（textarea）、截止时间选择器（QDateTimeEdit）、"创建"按钮。

**行为：**
- 填写完成后点击"创建"→ 任务出现在列表中，同网段设备可见

---

## S11 — 设置

**页面用途：** 应用设置。

**组件：**
- 常规：昵称编辑、下载目录、开机启动 toggle、最小化到托盘 toggle
- 网络：发现开关、监听端口、手动添加 IP

**行为：**
- 所有修改即时生效，写入 SQLite
- 端口修改后需重启网络服务（提示用户）

---

## S12 — 网络诊断

**页面用途：** 自助排查网络问题。

**组件：** 本机 IP 列表、监听端口状态、防火墙检测结果、UDP 广播测试按钮及结果、在线设备数。

**行为：**
- 页面打开时自动执行检测
- 点击"重新检测"刷新
- 每一项检测结果用绿色对勾或红色叉号表示

---

## S13 — 系统托盘菜单

**触发条件：** 右键系统托盘图标。

**菜单项：**
- 在线/隐身切换
- 打开主窗口
- 退出

**行为：**
- 在线/隐身：切换后停止/恢复 UDP 广播
- 退出：关闭所有连接，清理后退出进程

---

## S14 — 信息架构总览

参考原型图，页面间导航关系：

```
首次启动向导 ──→ 主窗口
                  ├── 消息（默认，含会话列表 → 聊天窗口）
                  ├── 联系人（在线设备列表 → 聊天窗口 / 手动添加IP）
                  ├── 传输中心（全部/进行中/已完成/失败）
                  ├── 共享文件
                  │   ├── 浏览共享（设备列表 → 目录浏览 → 下载）
                  │   └── 我的共享（添加/移除/启用禁用）
                  ├── 投递箱（列表 → 新建投递）
                  └── 设置
                      ├── 常规
                      └── 网络（发现/端口/手动IP → 网络诊断）
```

所有页面右上角保留通知区域（新消息/传输完成提示），不阻塞当前操作。

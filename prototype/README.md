# 烽燧 FengSui 高保真产品原型

打开 `index.html` 可离线查看完整高保真原型图集。

## 交付内容

- `index.html`：高保真原型查看器，包含页面索引、P0/P1 标签与源码状态标签。
- `screens/*.svg`：15 张 1280×820 高保真源图。
- `screens/*.png`：同名 PNG 导出图，便于放入评审文档、PRD、Wiki 或演示材料。
- `screens/manifest.json`：查看器使用的屏幕清单。

## 覆盖范围

原型覆盖 P0 主链路与 P1 关键屏：首次启动、消息、联系人、单聊文件发送、广播确认、传输中心、共享浏览、我的共享、授权弹窗、投递箱、设置、网络诊断、托盘菜单和信息架构总览。

## 设计口径

视觉风格采用稳重桌面工具方向，贴近 Qt Widgets 可落地的控件密度、间距、表格、标签页、按钮和弹窗形态。安全相关展示限定为首次授权、记住选择与共享访问控制，不展示当前 V1.0 文档禁止或源码未支持的端到端加密承诺。
